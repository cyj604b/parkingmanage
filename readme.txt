停车场管理任务分解

1 车辆进入停车场
	1.1 初始化停车场信息，如停车场数量、停车场容量、停车场停放车辆信息
		1.1.1 新建停车场类，属性：编号，容量，车辆信息；方法：得到停车场编号、容量、剩余车位、车辆信息，停车，取车
		1.1.2 新建车辆类，属性：车牌号；方法：得到车牌号
		1.1.3 实例化两个停车场，并设置每个停车场的容量、停放车辆信息
	1.2 选择停车场
		（1）比较每个停车场的空位数，得到空位数最大的停车场
		或
		（2）计算每个停车场的空置率（已停放车辆数/停车场容量），比较得到空置率最大的停车场
	1.3 停放车辆
		1.3.1 如上面得到的停车场为空，即每个停车场都满，则停车失败
		1.3.2 如上面得到的停车场不为空，车辆进入目标停车场，停车成功
2 车辆离开停车场
	2.1 司机进入某个停车场，出示停车凭证
	2.2 根据停车凭证，查询所取车辆是否在停车场内
	2.3如匹配成功，则取到车
	2.4如匹配失败，即车辆不在此停车场，则取车失败
3 停车仔打印停车场信息
	3.1得到所管理的所有停车场信息
	3.2循环每个停车场，读取停车场容量，剩余车位数，打印停车场编号，容量，剩余车位数
	3.3打印所有停车场车位数、剩余车位数
4 停车场经理打印停车仔和停车场信息
	4.1得到所管理停车仔信息
	4.2循环每个停车仔
	4.3循环停车仔对应停车场，读取停车场容量，剩余车位数，打印停车场编号，容量，剩余车位数
	4.4得到自己管理的所有停车场信息
	4.5循环每个停车场，读取停车场容量，剩余车位数，打印停车场编号，容量，剩余车位数
	4.6打印所有停车场车位数、剩余车位数
5 停车场主管打印停车场经理、停车仔、停车场信息
	5.1得到所管理停车场经理信息
	5.2循环每个停车场经理
	5.3得到停车场经理对应停车仔信息
	5.4循环每个停车仔
	5.5循环停车仔对应停车场，读取停车场容量，剩余车位数，打印停车场编号，容量，剩余车位数
	5.6打印停车仔所管理的总车位数和剩余车位数
	5.7打印停车场经理所管理的总车位数和剩余车位数
	5.8打印停车场主管所管理的总车位数和剩余车位数
